Kubernetes-v1.11.1-全自动ansible部署指南

发表于 2018-07-31

本文目标

  • 部署3节点的高可用master(v1.11.1目前最新版)+3个worker node 节点

  • 使用ipvs替代iptables
  • 容器部署flannel插件，coredns插件等相关插件
